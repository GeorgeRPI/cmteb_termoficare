# Integrare CMTEB pentru Home Assistant
### v.14.11.2025

Monitorizează starea sistemului de termoficare București direct în Home Assistant.

## Instalare

### Prin HACS
1. Deschide HACS -> Integrations
2. Click pe butonul cu 3 puncte -> Custom repositories
3. Adaugă: `https://github.com/numele_tau/cmteb_termoficare`
4. Instalează "CMTEB Termoficare Bucuresti"

### Manual
Copiază folderul `cmteb` din `custom_components/` în configurația ta Home Assistant.
